# MySimpleLandingPage


A personal landing page for **Srinivasa Gudi** built with plain **HTML, CSS, and JavaScript**.

## Overview

This is a single-page portfolio-style website that highlights:
- Personal intro and mission
- Featured and side projects
- School/community wins
- Timeline and long-term goals
- Contact links

## Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript** (no framework)

## Project Structure

- `/home/runner/work/MySimpleLandingPage/MySimpleLandingPage/index.html` — page content and sections
- `/home/runner/work/MySimpleLandingPage/MySimpleLandingPage/styles.css` — all styling and layout
- `/home/runner/work/MySimpleLandingPage/MySimpleLandingPage/script.js` — UI behavior (scroll header, nav highlighting, reveal animations)
